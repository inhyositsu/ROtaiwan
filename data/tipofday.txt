→/savechat  :  把對話視窗的內容儲存起來
#
→按住ctrl鍵，再按*鍵，可把微型地圖從半透明變成不透明，或是不透明變成半透明 
#
→按住shift鍵，再按滑鼠右鍵拖曳，可以把視角的角度抬高，如要恢復原狀，按住shift鍵，再按兩次滑鼠右鍵，既可恢復 
#
→按住ctrl鍵，再按滑鼠右鍵拖曳，可以把視角放大縮小，這時按住ctrl鍵，再按兩次滑鼠右鍵，既可恢復原狀，用滑鼠滾輪也可以把畫面放大縮小
#
→把游標放在敵人身上時，游標會變成刀， 既可攻擊
#
→從魔物那裡得到的經驗值，是與魔物受到的損害力同正比，
   ※一起攻擊魔物時，先攻擊魔物的人，得到的經驗值較多
#
→按住ctrl鍵攻擊魔物時，會變成[鎖定攻擊狀態]，將會持續自動攻擊到魔物死亡為止，想要解除自動攻擊狀態時，將滑鼠游標點選其他位置既可
#
→想要撿掉下來的物品，把游標指在物品上再[點選滑鼠左鍵]即可，撿到的物品會放在物品欄裡
#
→把游標移到物品身上[點選滑鼠右鍵]時，可以看到物品的情報
#
→玩家腳底下二個橫桿是玩家的HP值和SP值，綠色的表示HP值，藍色的表示SP值，HP值到0時，表示玩家已死亡，為了警惕玩家HP值剩下25%以下時，HP值會改變成顯示為紅色
#
→EXP顯示桿已滿的話(100%)，基礎等級會上升，基礎等級上升時，會獲得素質點數（Status Point），玩家可自由加給於想培養的素質
#
→EXP 表示桿下面的JOBEXP顯示桿已滿的話(100%)， 職業等級會上升，會獲得技能點數，玩家可自由加給於想培養的技能
#
→若玩家不動於原地時，HP和SP會稍微恢復
   ※坐著時會恢復速度會加快兩倍，而當重量超過50%以上時，將不會恢復HP值與SP值
#
→Weight(重量) : 顯示為最大負重量和現在攜帶的物品重量，玩家無法攜帶超過最大負重量的物品，物品重量超過最大負重量時，將不能攜帶任何物品。若攜帶的物品重量超過最大負重量一半時，將會顯示為紅色，這時HP和SP不會自動恢復，還有超過最大負重量90%時，玩家將不能攻擊和使用技能
#
→Zeny(金幣) : 金幣是在仙境傳說裡所使用的[金錢]單位 
   ※玩家所擁有的金幣當交易或購買物品時會自動扣減
#
→基礎等級上升後，會得到素質點數（Status Point）， 玩家可利用這點數培養素質
從1到10等級會得到3點數 ，每5等級會增加1點數
#
→STR(力量) : 對玩家攻擊力有直接的影響 ，力量增加時，最大負重量也將會增加，將可攜帶更多物品。力量增加將會增加近距離的攻擊力
#
→DEX(靈巧) : 對玩家使用武器時有直接的關係.
數值越高，越能發揮武器的攻擊力，用弓箭攻擊時，因DEX值提高也將會提高攻擊力，也會增加命中率，也影響到施放魔法的速度
#
→AGI(敏捷) : 對玩家的行動敏捷度有直接的關係.
增加這數值時，迴避率將會增加，可閃躲敵人的攻擊。而各種武器類的攻擊速度也將會提升.
#
→VIT(體質) : 對玩家的防禦力和HP值多寡有直接的關係.
   數值越高，防禦力也將會增加，遭受敵人攻擊時損失可減低，還有將會增加最大HP值和加快恢復HP的速度.
#
→INT(智力) : 對玩家的魔力（SP）有直接的關係，數值越高，將會增加魔法攻擊力（MATK）和魔法防禦力（MDEF）
#
→LUK(幸運) : 玩家的幸運對各種狀況都有直接的關係，主要是必殺攻擊率和完全迴避率，都和幸運增加有直接關係
#
→[Print Screen] or [Scroll Lock] : 拍照，照片會在[ScreenShot] 檔案內存檔起來
#
→[F10] : 可以調查對話視窗大小
#
→[F11] : 只保留基本情報視窗和對話視窗，關閉其他視窗
#
→[F12] : 可以開啟關閉快速使用列  
   ※將經常使用的物品或是技能拖曳於在F1∼F9鍵上，直接按所屬快速鍵就可以使用
#
→[INSERT] :將可坐下或站立  
※將[/坐下]或是[/站立]這兩個指令直接輸入於對話窗就可以使用。在坐下的狀態時HP、SP回復速度比站立時加快兩倍
#
→[Alt] + [End] : 可以把角色下方 [HP / SP] 圖示開啟/關閉
#
→[Alt] + [Home] : 可以把可移動的地區圖示開啟/關閉
#
→[Alt] + [=] : 切換輸入狀態，全形轉半形。用全形輸入的命令不會執行 這時候切換成半形輸入即可
#
→[Alt] + [F10] : 可以把聊天視窗開啟/關閉
#
→[Alt] + [M] :  讓各種表情符號 / 重複的聊天的文字，讓他重複的儲存。
#
→ [Alt] + [1~9] : [ALT] + [M]可以使用已儲存好的內容
#
→ [Ctrl] + [-,=,w]︰可以使用 剪刀 / 石頭 /布  表情符號
#
→/WHO : 可以看到伺服器上， 線上玩家總人數
#
→/WHERE : 可以看到現在所在位置的情報和座標
#
→/H : 顯示對基本的指令說明
#
→/BGM : 可以把背景音樂開啟/關閉  
   ※也可以按 [Alt] + [0] 在視窗上執行
#
→/SOUND : 可以把效果音開啟/關閉   
   ※也可以按 [Alt] + [0] 在視窗上執行
#
→/EFFECT : 可以把遊戲裡的各種效果圖示開啟/關閉   
   ※關閉效果時，可以使遊戲執行上更加順暢
#
→/SKIP :開啟/關閉畫面同步更新
   ※當畫面關閉時，因電腦配備不同會造成遊戲速度下降，當遊戲畫面繼續有Lag時建議你把他關掉
#
→/EX【角色名稱】:  可以隔絕此玩家所說的話和悄悄話
→/IN【角色名稱】:  恢復與此玩家之間的對話
#
→/EXALL : 把所有角色的對話隔絕
→/INALL : 恢復與所有玩家對話
#
→/EX : 顯示被隔絕玩家的清單
#
→/ORGANIZE 【隊伍名稱】: 把輸入的名字可以組成組隊員
   ※組隊後變成組隊長時，可以邀請玩家參加組隊
#
→把游標放在想邀請的玩家身上， 點選滑鼠右鍵，會顯示 ' 邀請參加組隊 ' 選項，利用這選項，可組成組隊 
#
→/LEAVE : 可以脫離自己參加的組隊 
   ※不會詢問 (Y/N)，所以要慎重考慮
#
→/EXPEL 【角色名稱】: 組隊長可將組隊員，從隊伍中刪除
   ※不會詢問 (Y/N)，所以要慎重考慮
#
→點選對話視窗右下段 [圓圈]後，將會顯示〝隊伍發言頻道〞，點選後將可使用隊伍頻道與隊員對談
#
→%  : 在發言前加上%既可直接使用隊伍頻道與隊員對談，或是按住Ctrl鍵，再按Enter鍵，把輸入的內容直接傳達給隊員  
#
→隊員右上方微型地圖將會顯示各隊員所在位置
#
→悄悄話 : 聊天視窗左邊輸入角色名稱，右邊輸入對話內容 ，用TAB鍵將可快速於二個輸入欄裡轉換位置
#
→選擇微型地圖 [+ / -] 符號時，地圖將會 [放大 / 縮小] 
#
→[MVP系統] : 與Boss級魔物對決時，為了保護最高傷害者的權益，魔物的MVP物品將會直接掉落於MVP玩家，如魔物未掉落MVP物品，將補償玩家50%的經驗值 
#
→遊戲在視窗模式進行時，可以移動視窗位置，按住[Alt] + [空白鍵]再按[M]鍵，可用^ff0000方向鍵或是滑鼠^000000調整位置
#
→/fog : 可以把霧化效果開啟/關閉，霧化效果是為各種地圖做天氣表現使用.
   對低階電腦配備玩家來說，霧化效果會增加電腦負擔，玩家如被迫跳出或停止遊戲時，這時建議玩家關掉霧化效果，霧化效果也可組態設定下永久開啟或是永久關閉
#
→從魔物掉下來的裝備用物品、物品效果不能檢定，可用道具商店所販賣的放大鏡來檢定
#
→獵人若要用陷阱系列技能時，身上必須要有《攜帶裝置用陷阱物品》
#
→組成工會需要消耗一個華麗金屬，組成工會指令為  /GUILD  [工會名字] ,工會名字不可有特殊字元與空白，用Alt +G 可以打開工會視窗
#
→ 當工會解散時， " /breakguild[[工會名字] " 就可以解散工會，但工會員只剩下工會長1名時才可以使用
#
→工會會員之間聊天可使用 /gc 【對話內容】 或是 $ 【對話內容】 或是按 Alt + Enter 也可以
#
→想要組合的物品，放在組合視窗上點選卡片兩次就可以組合，一定要把組合的物品，放在組合視窗上
#
→對 幾種異常狀態效果說明
  昏迷 : 變成昏迷狀態，將不能行動
  黑暗 : 畫面變黑暗，對魔物來說視線會變狹礙
  冰凍 : 將無法行動，當受到攻擊時，才會解除此狀態
#
→死亡時經驗值會下降，使用技能後或是戰鬥狀態後，需等待5秒才可下線
#
→使用倉庫，紀錄儲存點，租用手推車，空間移動等，詢問卡普拉職員即可
#
→[Alt] + [J] : 可以看到現在自己擁有的寵物情報
#
→角色追蹤技能 : 按[Shift] + [滑鼠右鍵]點選所想追蹤的角色，便可以追蹤角色
#
→[Alt] + [L] : 顯示出現在可以使用的表情符號目錄，可以把它移動到快捷列上
#
